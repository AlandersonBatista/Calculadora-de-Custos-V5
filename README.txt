# Calculadora de Custos (PWA) — v5 com WhatsApp e foto

Novidades desta versão:
- Botão "Tirar/selecionar foto": usa a câmera ou galeria do celular.
- Botão "Enviar cálculo para WhatsApp":
  - Monta uma mensagem com FOB, CONVERSÃO, FATOR, CUSTO, PDV, IMPOSTOS, CUSTO FINAL e MARGEM.
  - Em navegadores que suportam a Web Share API com arquivos (ex.: Chrome Android), envia a FOTO + TEXTO.
  - Caso contrário, abre o link wa.me com o texto (sem foto).

Observações:
- Ainda é necessário estar online para enviar a mensagem pelo WhatsApp.
- O funcionamento exato pode variar entre Android/iOS e entre navegadores.
